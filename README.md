# Excel Sales Dashboard

## Overview

This project presents an interactive sales dashboard created using Microsoft Excel to analyze sales performance and support business decision-making.

## Dashboard Features

The dashboard provides:

- Total sales amount by company
- Sales by product
- Sales by sales representative
- Monthly sales by product
- Sales by state and company
- Sales by state

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Visualization
- Dashboard Design
